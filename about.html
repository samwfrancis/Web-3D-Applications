About

For my website, I wanted to make an online record store to display my passion for music and vinyl. Records seemed like a great way to add my own 3D model to the website as well so I made three different variants of the record.


3D Models

I have made 3 3D models which are different variants of vinyl. Due to the shader I used to create the vinyl grooves, it proved hard to make implement into the website. The record used in the model viewer is an altered version made to work on my website however, the original version will be below.


Design

The top and bottom banners for the website were done using the bootstrap navbar. To allow for a larger selection of records, I used the bootstrap carousel which was modified to display three records at a time. The records in the carousel used the zoom hover bootstrap style to highlight the record that your mouse is over. Using javascript, I added a container for my 3D model which includes lighting, navigation and the ability to toggle the wireframe and swap between models. It also includes a turntable animation to simulate the record being played. Below is the html for the carousel which is my favourite part of the website.


Implementation

As there were so many records added to the store it wasn't feasable for me to create individual store pages for all of them, however, there are store pages for Kind of Blue, Voodoo, Dark Side of the Moon, Dummy, Random Access Memories and What's going on. These pages demonstrate what the rest of the records would look like as they follow a similar template.


Feedback

All feedback from users was positive, especially on the layout of the cite and the colour choices which didnt distract from the album covers.


Deeper Understanding

I feel that i have fully understood the material that was covered in the Labs as well as adding extra components to my website like the carousel and hover zoom.
